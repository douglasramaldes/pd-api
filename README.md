#API

- NodeJS
- Web Framework - Fastify
- Database - Mysql

#Requerimentos

- node > 8.XX
- npm > 6.XX

#Instalação

```bash
npm install

'Substitua as variáveis DB_USERNAME e DB_PASSWORD no file .env que encontra-se no root do projeto com as suas informações de usuário do Mysql Local'

npm run seed

npm run dev
```
